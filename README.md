# KGL - Karibu Groceries Limited Management System

A comprehensive agricultural produce management system for managing inventory, sales, and operations across multiple branches.

##  System Overview

KGL system manages two branches (**Maganjo** and **Matugga**) with:
- **2 Managers** (1 per branch)
- **4 Sales Agents** (2 per branch)

##  Features

- **User Authentication & Authorization**
  - Secure JWT-based authentication
  - Role-based access control (Manager, Sales Agent)
  - Branch-specific user management
  
- **Manager Dashboard**
  - Stock procurement management
  - Price management
  - Sales tracking
  - Credit sales monitoring
  - Comprehensive reports
  
- **Sales Agent Dashboard**
  - Make sales
  - View stock availability
  - Check current prices
  - Track personal sales history

##  Prerequisites

- Node.js (v14 or higher)
- MongoDB (v4.4 or higher)
- npm or yarn

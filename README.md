## Karibu Groceries Limited
- This is a KGL website built with html and bootstrap 5

## Folder
- **Auth**: This contains the authentication Pages namely; registration and Login Page

- **Manager**: This contains the managers dashboard and managers pages like credit sales,prices,procure,reports,sales and stock 

- **SalesAgent**: This contains the sales agent dashboard and salesagent pages like makesale, sales,prices and stock

- **Director** : This contains the directors dashboard (view only) and KGL's reports
